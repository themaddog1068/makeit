# Hood Ornament — Base Geometry

4-color badge/emblem designed for Anycubic Kobra 3 Max with ACE system.

## Color layout (ACE slots)

| Slot | File | Description |
|---|---|---|
| 1 | `black_base.stl` | Full base plate, 188×92mm, 1.2mm thick |
| 2 | `white_teeth.stl` | Teeth (mouth row + lower row) and both side fangs |
| 3 | `green_face.stl` | Main face/head body |
| 4 | `yellow_eyes.stl` | Eye inserts, two symmetrical pieces |

## Ready-to-slice file

`hood_ornament_v3.3mf` — load directly in Anycubic Slicer Next, all 4 parts pre-colored.

## Source files

Derived from `nested_locked_outline_layers.zip` (locked outline layer set).
Shell extraction from green layer:
- **White**: shells 3, 4, 7, 8, 9, 13, 15, 16
- **Removed** (black base shows through): shells 5, 6, 10, 11, 12, 14
